
# Webpack Template

### Clone the repo
```
git clone git@github.com:IstiakAR/Webpack-Template.git
```

### CD into the directory and Install all the dependencies
```
npm install
```
### Run the project
```
npx webpack serve
```
It will be running on http://localhost:8080/

### To start a github page

```
npx webpack --mode production
```

```
npm run deploy
```
